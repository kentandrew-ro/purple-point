# purple-point

IM2 Proposal

## Database documentation

Database documentation and the ER diagram are generated by
[`tbls`](https://github.com/k1LoW/tbls) using `tbls.yaml`.

After loading the project environment variables, generate the documentation:

```sh
source .envrc
make erd
```

The generated Markdown and SVG files are written to `docs/schema/`. To apply
`database.sql` and then regenerate the documentation in one command, run:

```sh
make schema
```

The database password used in the DSN must be URL-encoded if it contains
characters such as `#`, `<`, `@`, or `:`.
